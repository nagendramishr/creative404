# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-12

### Added
- Complete .NET 10 Blazor Server project structure
- Interactive 404 page generator with Blazor components
- Random message and color generation
- Home.razor page with real-time interactivity
- NotFound.razor page as example 404 page
- CSS isolation for component-specific styling
- Responsive design with gradient backgrounds
- Navigation menu with Generator and Example pages
- Comprehensive documentation for .NET/Blazor
- Project configuration files (appsettings.json, launchSettings.json)
- Blazor Server with SignalR for real-time updates

### Changed
- Migrated from HTML/CSS/JavaScript to .NET 10 Blazor Server
- Updated README with .NET-specific instructions
- Updated documentation for Blazor architecture
- Modernized .gitignore for .NET projects
